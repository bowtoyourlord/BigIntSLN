Проект являє собою лише два файли - BigInt.h та BigInt.cpp (main.cpp - зручність та використовується для демонстрації.)
На GitHub він був завантажений напряму з Microsoft Visual Studio 2022.
Особливого компілювання проект не потребує - бо фактично є консольним, і запускається просто клавішою F5.
Запускати з-під файлу main.cpp.
